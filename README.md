# オノコムメンバーズ サービス仕様書

株式会社オノコムが提供する「オノコムメンバーズ」は、AWS利用料連動モデルから脱却し、お客様の多様なニーズに応えるための新サブスクリプションプランです。

## サービス概要

オノコムメンバーズは、従来の「守り」のサポートから「攻め」のIT活用へとシフトし、お客様のビジネス成長を加速させるパートナーサービスです。

### 解決する課題

- **「何を頼めばいいか分からない」を解消**  
  月額1万円のプランで、課題整理から伴走します
- **必要な時に、必要なだけ専門家の力を**  
  短期集中型の「ブーストオプション」で、無駄なコストを削減
- **守りから「攻め」のIT活用へ**  
  ビジネスの成長を加速させるパートナーとして、お客様の挑戦を支援します

### サービスの特長

- AWS公認パートナーネットワーク（APN）参加企業による専門的支援
- 2025 Japan All AWS Certifications Engineers（全認定資格取得者）を含む認定技術者が在籍
- AI・クラウド導入から業務部門との内製化支援まで幅広くサポート
- PoC（概念実証）立ち上げをお客様と共に進める伴走型サービス

## サービス仕様

### 基本仕様

| 項目 | 内容 |
|------|------|
| **コミュニケーション** | 当社ポータルサイト、Slack等の指定チャットツール |
| **対応時間** | 平日10:00〜17:00（時間外の質問への返信は翌営業日以降） |
| **返信速度** | ベストエフォートとし、SLA（品質保証）は設けていません<br>定例会がある場合、まとめて回答となる場合もあります |
| **質問回数** | プランに応じて月5インシデントまでなど制限があります |
| **相談内容** | クラウド・AIに関するQ&A、一般的なベストプラクティスに関する助言（AWS以外のクラウドサービスも対象） |

### 対象外サービス

以下のサービスは対象外となります：
- 障害の調査、切り分け、復旧作業
- 即時応答を要する緊急対応
- 技術仕様外の業務相談

※ただし、課題として事前に定義されている場合はこの限りではありません

### 契約条件

- **初月料金**: 無料
- **解約**: 当月中の申し出で当月末に解約
- **日割り計算**: 解約月の日割り計算はございません
- **AWSディスカウント**: AWSクラウドの契約において割引上限があり、メンバーズ加入においてもその上限を超えることはできません

## 従来サポートとオノコムメンバーズの違い

従来の「守り」のサポートから「攻め」のメンバーズへ

> **サポート** = 突発的な問題に対応する救急救命医  
> **メンバーズ** = 計画的に未来を作るパーソナルトレーナー

### 比較表

| 項目 | 従来のサポート | オノコムメンバーズ |
|------|--------------|-----------------|
| **主役（対象）** | システム・AWSアカウント（モノ） | お客様のチーム・ビジネス（ヒト・コト） |
| **担当** | AWS認定者チーム | 担当制 |
| **役割** | 請負業者 | パートナー |
| **スタンス** | 守り | 攻め |
| **主な活動** | 決まった手順の運用・保守 | 未知の可能性を探る概念実証（PoC） |
| **アプローチ** | 建築的アプローチ<br>（最初に完璧な設計図を作り、その通りに建てる） | ソフトウェア的アプローチ<br>（作りながら試し、改善を繰り返して進化させる） |
| **提供価値** | 安定稼働、品質維持 | 事業成長、イノベーション促進 |
| **ゴール** | 決められた仕様を完璧に実現する | ビジネスの可能性を最大化する |
| **SLA** | あり（障害検知後の一次応答時間、<br>復旧目標時間などを具体的に規定） | 原則なし（定例会の実施やQ&Aへの応答は<br>ベストエフォートとし、具体的な応答時間は保証しない） |
| **障害への対応** | 回復を第一優先とする | 範囲外とする<br>（障害対応の最適化などを課題として取り組むことは可能） |

## ブーストオプション詳細

メンバーズプラン専用の短期集中支援パック

### 概要
- **料金**: ¥200,000〜/月（まとめて複数月をご契約）
- **最低契約期間**: 1ヶ月〜
- **対象**: メンバーズプランご契約のお客様

### 提供サービス

#### 作業支援の大幅拡張
- **月20時間〜の作業支援**
- 高度な技術課題への対応
- 認定技術者による直接サポート

#### PoC計画・実行支援（認定者対応）
- AWS全認定資格取得者による計画策定
- 実際の環境構築支援
- 検証結果の評価・改善提案

#### セキュリティレビュー（認定者対応）
- セキュリティ専門認定者による詳細レビュー
- AWS Well-Architectedフレームワークに基づく評価
- 具体的な改善提案とその実装支援

#### 対面定例会
- **月1回〜の対面での定例会**
- プロジェクトの進捗確認
- 次期計画の策定
- 直接的なコミュニケーションによる課題解決

### 適用場面
- 特定のプロジェクトを短期集中で加速させたい
- 認定技術者の高度な支援が必要
- 対面でのコミュニケーションを重視したい
- 月5時間では不足する作業支援が必要

## よくある問題とオノコムメンバーズによる解決策

オノコムメンバーズは、以下のような問題を解決します：

### 🤔 契約したいが何をどのように契約すればよいかわからない
**解決策**: カジュアルプラン（月額1万円）で課題整理から伴走し、最適なサービス選択をサポートします。

### 📋 委託する内容を取り決める前段階で課題が発生している
**解決策**: プロジェクト開始前の課題整理・要件定義から専門家が支援し、適切な方向性を見つけます。

### 💬 発注しないのに相談だけしてもよいか
**解決策**: サブスクリプション型のため、発注を前提としない相談・アドバイスが可能です。

### 🌐 当社より提供外のAWSクラウドやAIのことについて質問したい
**解決策**: AWS・AI全般に関する質問にお答えします。特定の製品・サービスに限定されません。

### 💰 定額制でサービスを提供してほしい
**解決策**: 月額固定料金制のため、予算計画が立てやすく、コストコントロールが可能です。

### 🛠️ 構築は自分たちでやるからやり方だけ教えてほしい
**解決策**: 内製化支援として、ベストプラクティスの共有や技術指導を行います。

### 新しいパートナーシップが必要な理由

現代のIT環境では以下の変化が起きています：

1. **開発アプローチの変化**  
   完璧なシステムを最初に設計・構築するスタイルから、ビジネスの変化に合わせて柔軟に改善を繰り返すアプローチが主流に

2. **クラウドの成熟**  
   インフラ担当者の大きな役割だった「稼働の担保」は、その大半をAWS自身が担うようになり、私たちの介在価値も変化

3. **お客様からの期待の変化**  
   「問題を完璧になくす」ことから「ビジネスを成長させる新しい課題を発掘し、共に挑戦する」ことへとシフト
